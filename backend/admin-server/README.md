# admin-server

## 목적
관리자 API, Spring Boot Admin Server, 모니터링 및 감사 로그 조회 기능을 제공한다.

## 주요 책임
- 관리자 인증/인가
- 사용자 관리 API
- 권한 관리 API
- 감사 로그 조회 API
- 시스템 상태 조회
- Spring Boot Admin Server

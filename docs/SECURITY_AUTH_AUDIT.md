# SECURITY_AUTH_AUDIT

## 1. 인증
- ID/PW 필수
- 비밀번호 단방향 해시 저장
- 로그인 실패 5회 계정 잠금
- 세션 만료 30분
- 로그인 성공/실패 감사 로그 기록

## 2. 인가
- RBAC 적용
- 모든 보호 API 서버 권한 검증
- 관리자 기능은 ADMIN/AUDITOR 권한만 허용

## 3. 감사 로그
필수 기록 대상:
- 로그인/로그아웃
- 사용자 등록/수정/삭제
- 권한 변경
- 관리자 설정 변경
- 감사 로그 조회
- 권한 없는 접근 시도

## 4. OWASP 기준
- SQL Injection 방지
- XSS 방지
- CSRF 방지
- 접근통제 서버 검증
- 취약 컴포넌트 사용 금지

## 5. 로그 금지 항목
- 비밀번호
- 토큰
- 세션 ID
- API Key
- 주민등록번호
- 계좌번호
- 카드번호

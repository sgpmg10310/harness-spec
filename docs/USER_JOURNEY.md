# USER_JOURNEY

## 1. 사용자 흐름
로그인 → 메인 → 목록 조회 → 상세 조회 → 등록/수정 → 로그아웃

## 2. 관리자 흐름
관리자 로그인 → 대시보드 → 사용자 관리 → 권한 관리 → 감사 로그 조회 → 시스템 모니터링

## 3. 예외 메시지
| 상황 | 메시지 |
|---|---|
| 로그인 실패 | ID 또는 비밀번호를 확인해주세요. |
| 계정 잠금 | 로그인 실패 횟수 초과로 계정이 잠겼습니다. |
| 권한 없음 | 해당 기능에 접근할 권한이 없습니다. |
| 저장 성공 | 저장이 완료되었습니다. |
| 서버 오류 | 일시적인 오류가 발생했습니다. 관리자에게 문의해주세요. |
